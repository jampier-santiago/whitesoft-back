# Pasos para empezar

1. Clonar el repositorio
2. Ejecutar

```
npm install
```

3. Llenar las variable de entorno definidas en el **.env**

- MONGODB_ATLAS=MONGODB_ATLAS
- SECRET_KEY=SECRET_KEY
- API_COUNTRIES=API_COUNTRIES

4. Ejecutar la aplicacion en dev:

```
npm run start:dev
```

# Stack usado

- Node ts
- Mongo db
- Express
